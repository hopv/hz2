
From example 2.3

* `ex2_3.target` : HORS and APT
* `ex2_3_man.v` : Generated Coq code with a proof without our tactics
* `ex2_3_tac.v` : Generated Coq code with a proof with our tactics
* `ex2_3_tactics.v` : Generated tactics loaded from above Coq files

The .v files above can be automatically generated by running:
```
hz2 -i ex2_3.target
```
